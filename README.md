# 🚗 Car Showroom

A modern React + Vite application for browsing and reviewing cars with a
clean UI built using Material UI.

#Vercel link

https://carshowroom-rose.vercel.app/ 

------------------------------------------------------------------------

## ✨ Features

-   📦 Built with Vite for fast development
-   ⚛️ React 19
-   🎨 UI with Material UI (MUI)
-   🔄 Routing with React Router
-   🌐 API requests via Axios
-   💾 Local storage support (reviews)
-   ⭐ Rating system for cars
-   📝 Add reviews
-   📱 Responsive design

------------------------------------------------------------------------

## 🛠️ Tech Stack

-   React
-   Vite
-   Material UI (MUI)
-   Emotion & Styled Components
-   React Router DOM
-   Axios
------------------------------------------------------------------------

## ⚙️ Requirements

- Node.js version: **22.21.1**

------------------------------------------------------------------------

## 📦 Installation

Clone the repository:

``` bash
git clone https://github.com/your-username/car_showroom.git
cd car_showroom
```

Install dependencies:

``` bash
npm install
```

------------------------------------------------------------------------

## ▶️ Running the App

Start development server:

``` bash
npm run dev
```

Build for production:

``` bash
npm run build
```

Preview production build:

``` bash
npm run preview
```

## 💾 Local Storage

Reviews are stored in the browser using `localStorage`:

-   Key: `reviews`
-   Automatically persisted after submission

------------------------------------------------------------------------

## 🔗 Routing

Managed via `react-router-dom`:

-   `/` --- Home page
-   `/cars` --- Car listing
-   `/cars/:id` --- Car details page
